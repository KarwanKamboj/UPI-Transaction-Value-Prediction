# UPI Transaction Value Prediction

## Project Objective
To predict UPI transaction value using RBI Digital Payments dataset by applying supervised machine learning (Multiple Linear Regression).

## Dataset
RBI Digital Payments Dataset containing daily transaction values for UPI, NEFT, RTGS, AEPS, and card-based payment systems.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
